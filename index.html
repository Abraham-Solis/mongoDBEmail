Product Updates

Data densification for time series

Although time series data often has gaps (from an IoT sensor failure, for instance), continuous data is crucial for analytics, aggregations, and other operations. To get around this issue, MongoDB 5.3 now offers gap filling. When used in tandem with densification, these two features can help you readily handle missing data — and better surface valuable insights.

Realm Flutter SDK – now with Windows support!

Seamlessly integrate Realm into all of your Flutter projects – regardless of version or platform – with the latest release of the Realm Flutter SDK (Alpha). This release also includes an official integration with the Dart FFI APIs, marking the next step forward in our journey to Beta and GA.

Check out the documentation or our announcement post to learn more.
